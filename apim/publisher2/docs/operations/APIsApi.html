Change API Status

Change the lifecycle of an API

URL

https://apis.wso2.com/api/am/publisher/v0.9/apis/change-lifecycle

HTTP Method

POST

Parameters

Type Name Description Required Schema Default
Query action

The action to demote or promote the state of the API.\n\nSupported actions are [ **Publish, Deploy as a Prototype, Demote to Created, Demote to Prototyped, Block, Deprecate, Re-Publish, Retire **]

 String
Query apiId

**API ID** consisting of the **UUID** of the API. \nThe combination of the provider of the API, name of the API and the version is also accepted as a valid API I.\nShould be formatted as **provider-name-version**.

 String
Query lifecycleChecklist

You can specify additional checklist items by using an **\"attribute:\"** modifier.\n\nEg: \"Deprecate Old Versions:true\" will deprecate older versions of a particular API when it is promoted to \nPublished state from Created state. Multiple checklist items can be given in \"attribute1:true, attribute2:false\"\nformat.\n\nSupported checklist items are as follows.\n1. **Deprecate Old Versions**: Setting this to true will deprecate older versions of a particular API when it is promoted to Published state from Created state.\n2. **Require Re-Subscription**: If you set this to true, users need to re subscribe to the API although they may have subscribed to an older version.

 String
Header If-Match

Validator for conditional requests; based on ETag.

 String
Header If-Unmodified-Since

Validator for conditional requests; based on Last Modified header.

 String

Responses

HTTP Code Description Schema

200

OK. \nLifecycle changed successfully.

400

Bad Request. \nInvalid request or validation error

 Error

404

Not Found. \nRequested API does not exist.

 Error

412

Precondition Failed. \nThe request has not been performed because one of the preconditions is not met.

 Error
